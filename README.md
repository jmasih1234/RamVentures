# Ram Ventures

**Building the entrepreneurial ecosystem at Colorado State University**

Ram Ventures is an initiative focused on developing a unified startup ecosystem at CSU. This repository contains the MVP platform that powers venture discovery, collaboration, and early-stage incubation across the Northern Colorado (NoCo) region.

---

## The Problem

Student entrepreneurs often face:
- Fragmented resources (events, funding, mentorship across different platforms)
- Difficulty finding collaborators and co-founders
- Limited visibility into active startups and projects
- No centralized system connecting students, alumni, and investors

---

## 💡 Our Solution

Ram Ventures is building a **connected entrepreneurial ecosystem**, starting with a platform that:
- Centralizes student startups and senior design projects
- Enables collaboration across disciplines
- Provides structured pathways for different types of builders
- Bridges the gap between students, alumni, and early-stage investors

---

## Features (MVP)

- 🔎 **Ventures Directory**  
  Browse startups with filters (skill, major, stage)

- 📄 **Project Pages**  
  View venture details, open roles, and apply/contact

- 🧭 **User Pathways**  
  Tailored journeys for:
  - Explorer
  - Entrepreneur
  - Civic
  - Creative
  - Intrapreneur

- 🤝 **Mentor & Investor Signup**  
  Alumni engagement through structured onboarding

- 📅 **Events & Demo Days**  
  Centralized listings for ecosystem activity

- ⚙️ **Admin Dashboard**  
  Create and manage ventures and events
